# ข้อเสนอธุรกิจอีคอมเมิร์ซ: ร้านเครื่องประดับเพชรสังเคราะห์ (Lab-Grown Diamond) สำหรับใส่ได้ทุกวัน

### 1. Human Problems (ปัญหาของผู้บริโภค)
* เพชรแท้มีราคาสูงเกินไป ทำให้นักศึกษาและคนทำงานรุ่นใหม่เข้าถึงยาก
* มีความกังวลเรื่องความปลอดภัยหากต้องใส่เครื่องประดับเพชรแท้ในชีวิตประจำวัน
* ขาดทางเลือกที่เป็นเกรดพรีเมียมในราคาจับต้องได้ (เพชรแฟชั่นทั่วไปมักจะขุ่นมัวง่าย)

### 2. Solution
* นำเสนอเพชรสังเคราะห์ (Lab-Grown Diamond) ที่สวยประกายเหมือนเพชรแท้ แต่ราคาถูกกว่า 70-80%
* ออกแบบสไตล์มินิมอล ทนทาน เหมาะกับการแต่งตัวไปเรียนหรือไปทำงานทุกวัน (Everyday Wear)
* สวยได้แบบไม่ต้องกังวลเรื่องสูญหาย และเป็นมิตรต่อสิ่งแวดล้อม 100%

### 3. Core Features
1. Browse Products (ระบบเลือกดูสินค้าแยกตามประเภท)
2. Add to Cart (ระบบเลือกไซส์/สีตัวเรือน แล้วเพิ่มลงตะกร้า)
3. Place Order & Pay (ระบบสั่งซื้อและแนบหลักฐานชำระเงิน)
4. Admin Manages (ระบบหลังบ้านสำหรับแอดมินจัดการสต็อกและอัปเดตสถานะออเดอร์)
